# SPALF: Spectrally-Preconditioned Augmented Lagrangian on Stratified Feature Manifolds

Standard sparse autoencoders (SAEs) minimize a weighted sum of reconstruction error and sparsity. This requires manually tuning the weight between the two objectives, and the right setting depends on both the data and the optimizer. SPALF removes this tuning by treating SAE training as constrained optimization: minimize $\ell_0$ sparsity subject to faithfulness, lexical drift, and orthogonality constraints, solved via an augmented Lagrangian with adaptive dual updates.

**Stratified decoder.** The decoder has two blocks. An *anchored* block $W_A \in \mathbb{R}^{d \times V}$ is initialized from the model's unembedding matrix and kept aligned with vocabulary directions through a drift constraint, giving each anchored feature a direct lexical interpretation. A *free* block $W_B \in \mathbb{R}^{d \times (F-V)}$ is learned from scratch under unit-norm projection to capture non-lexical structure. The anchored encoder rows are set to whitened vocabulary vectors $W_{\text{enc}}^{[:V]} = (\Sigma^{-1/2} W_{\text{vocab}})^\top$ so that each pre-activation measures similarity in the precision-weighted (Mahalanobis) inner product. The free encoder rows are QR-orthogonalized against the anchored subspace, and the decoder bias is set to the data mean.

**Smooth JumpReLU gating.** Features are gated by JumpReLU with learnable per-feature log-thresholds. The step function in JumpReLU is non-differentiable, so during backpropagation it is replaced by a smooth quadratic surrogate: within a transition zone $(-\sqrt{2\gamma_j}, 0]$ around each threshold, the gradient is a linear ramp $-u/\gamma_j$, giving a continuously differentiable approximation to the Heaviside function. The per-feature bandwidth $\gamma_j$ is set from the interquartile range of pre-activations near each threshold. Unlike the piecewise-constant kernel used in DeepMind's JumpReLU SAE ([arXiv:2407.14435](https://arxiv.org/abs/2407.14435)), this surrogate has no gradient discontinuity at the decision boundary. Both the $\ell_0$ objective and constraint violations receive gradients through this surrogate via fused Triton kernels.

**Constrained augmented Lagrangian.** The augmented Lagrangian uses a smooth penalty $\Psi(g, y) = (\max(0, 2g+y)^2 - y^2)/4$ that transitions continuously between active and inactive constraints, avoiding the non-smooth transition of the standard quadratic penalty ([arXiv:2510.20995](https://arxiv.org/abs/2510.20995)). Dual variables $\lambda$ are updated using both the current constraint violation and its change from the previous step: $\lambda \leftarrow [\lambda + \rho(2v - v_{\text{prev}})]_+$. Setting the proportional gain equal to the penalty weight $\rho$ makes this equivalent to gradient descent-ascent on the augmented Lagrangian ([arXiv:2509.22500](https://arxiv.org/abs/2509.22500)). Each constraint has its own adaptive penalty weight that can both increase and decrease: it grows when violations are large relative to their running variance, and shrinks when constraints are satisfied, avoiding over-penalization in late training ([arXiv:2412.14269](https://arxiv.org/abs/2412.14269)). A per-constraint lower bound on the penalty weight ensures convergence. Constraint violations are smoothed with a triple exponential moving average (3·EMA$_1$ − 3·EMA$_2$ + EMA$_3$) that eliminates first- and second-order tracking bias, reducing lag by a factor of $(1-\beta)^2$ relative to a single EMA with no additional hyperparameters ([arXiv:2306.01423](https://arxiv.org/abs/2306.01423)). Dual updates are deferred when constraints are feasible and improving, recovering exact KKT conditions asymptotically.

**Residual-coupled annealing.** A single scalar $r(t) \in [0, 1]$ measures how far training is from satisfying the constraints and drives three coupled schedules: (1) the surrogate bandwidth shrinks as $\gamma = \max(r, \alpha_{\text{floor}}) \cdot \gamma_{\text{init}}$, making the surrogate approach the true step function as constraints are met, with $\alpha_{\text{floor}} = (d/F)^2$ preventing excessive gradient amplification and a time-dependent floor $\gamma_{\text{floor}}(t) = O(t^{-1/3})$ ensuring the bandwidth does not decay faster than the dual variables can track ([arXiv:2003.07612](https://arxiv.org/abs/2003.07612)); (2) a discretization coefficient $\lambda_{\text{disc}} = \max(1 - r, \lambda_{\text{disc}}^{\text{prev}})$ grows monotonically to close the gap between the surrogate $\ell_0$ and the true $\ell_0$; (3) once all reconstruction constraints are satisfied, a KL-divergence constraint activates via a cumulative-sum change-point detector, shifting the fidelity target from whitened reconstruction error to the distributional match between original and SAE-patched next-token predictions.

**Spectral whitening.** Activations are whitened online using a streaming low-rank covariance sketch ([arXiv:1501.01711](https://arxiv.org/abs/1501.01711)) that maintains an $O(\sqrt{d} \times d)$ approximation via repeated SVD compression, followed by ZCA whitening with per-eigenvalue shrinkage optimized under the Frobenius loss for numerically stable inversion. The whitener provides the Mahalanobis metric used in both the faithfulness constraint and the encoder initialization.
